## AUScript v1.0 - A little language with big attitude

Welcome to **AUScript** — a homemade interpreted language where everything is an expression, and the syntax feels like C's younger sibling that actually behaves.

### What's inside

- **Interactive calculator mode:** type an expression, get the result instantly.
- **Variables** that spring into existence on first assignment. No ceremony.
- **Arrays** with dynamic sizing — just like you'd expect.
- Full arithmetic, comparison, boolean logic, and math functions (`sin`, `cos`, `sqrt`, `^`, `#` (square root)).
- Convenient **control flow:** `if`/`else`, `while`, `for`, plus `break` and `continue` that actually work across nested loops.
- Built-in I/O: `scan` from stdin, formatted output with `printw`, and the `echo` command for quick outputs.
- Under the hood: **Dijkstra's shunting-yard algorithm** for expression parsing, recursive descent for control structures. Classic compiler techniques, no magic.
- Comes with examples: multiplication table, decimal-to-binary converter, insertion sort, and even a Leibniz series for pi with Euler transformation.
